# Dashboard-Ev
1.	Research Objectives
1.	Develop an interactive dashboard for EV charging load analysis. 
2.	Simulate EV charging behavior using Monte Carlo methods. 
3.	Evaluate the impact of EV charging on transformer loading. 
4.	Analyze different charging scenarios: 
o	Residential 
o	Office 
o	Public 
5.	Provide visual decision-support tools for planners and utilities.
2.	What the Dashboard Should Show
Case 1: Residential
•	Peak load profile 
•	Peak charging hour 
•	Transformer utilization (optional for user )
Case 2: Office
•	Daytime charging impact 
•	Transformer loading 
Case 3: Public
•	Random charging behavior 
•	Peak demand variation 
Case 4: Mixed Scenario (testing )
•	Residential + Office + Public combined
